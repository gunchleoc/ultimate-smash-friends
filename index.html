Fight!

Ultimate Smash Friends is a 2d arcade fight game, inspired by the nintendo super smash bros series. It's written in python with pygame and thus is crossplatform (Windows, MacOS, Linux, and probably others). It features original characters and levels. As a free software project, it encourage improvements and extensions by the users wanting to make it better, design, art, code and other contributions are all welcome!

The project is aimed at being fun, nice, and original in realisation.

The game currently allows up to four players to play on the same computer. There is a limited Artificial intelligence, and still no network game (any expertise with that appreciated).
